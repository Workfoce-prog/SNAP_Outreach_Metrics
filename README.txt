
# SORM - SNAP Outreach Metrics Streamlit App

This app helps organizations identify and prioritize households likely eligible for SNAP,
especially when formal outreach programs are defunded. It calculates key scores:
- Eligibility Likelihood
- Outreach Priority
- Food Insecurity Risk
- Enrollment Support Needs

## To run locally:
```
pip install streamlit pandas plotly
streamlit run app.py
```
